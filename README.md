# react-aviasales-app

# Installation

```sh
git clone https://github.com/TENEr98/react-aviasales-app.git
cd react-aviasales-app
```

# Configuration

Create file .env.local Paste Token and Base Url into file

- VITE_TOKEN_KEY=
- VITE_BASE_URL=

# Run

```sh
yarn
yarn dev
```

# Build

```sh
yarn build
```
